## Version control

- Quick commits with VS Code's built in [[git-integration]]
- Store your workspace in an auto-synced GitHub repo with [Write your notes in GitHub Gist](/user/recipes/write-your-notes-in-github-gist)
- Sync your GitHub repo automatically using the [GitDoc VSCode Plugin](https://marketplace.visualstudio.com/items?itemName=vsls-contrib.gitdoc) [Automatically Sync with Git](/user/recipes/automatic-git-syncing).

## Publish

- Publish using official Foam template
  - Publish to [GitHub Pages](/user/publishing/publish-to-github-pages)
  - Publish to [GitLab Pages](/user/publishing/publish-to-gitlab-pages)
  - Publish to [Publish to Azure DevOps Wiki](/user/publishing/publish-to-azure-devops-wiki)
  - Publish to [Publish to Vercel](/user/publishing/publish-to-vercel)
- Publish using community templates
  - [Publish to Netlify with Eleventy](/user/publishing/publish-to-netlify-with-eleventy) by [@juanfrank77](https://github.com/juanfrank77)
  - [Generate a site using Gatsby](/user/publishing/generate-gatsby-site) by [@mathieudutour](https://github.com/mathieudutour) and [@hikerpig](https://github.com/hikerpig)
  - [Generate a site using the Material for MkDocs theme](/user/recipes/generate-material-for-mkdocs-site) by [@djplaner](https://github.com/djplaner)
- Make the site your own by [Publish to GitHub](/user/publishing/publish-to-github).
- Render math symbols, by either
  - adding client-side [Math Support](/user/publishing/math-support-with-mathjax) to the default [GitHub Pages](/user/publishing/publish-to-github-pages) site
  - adding a custom Jekyll plugin to support [Katex Math Rendering](/user/publishing/math-support-with-katex)
- Export note to PDF [Export to PDF](/user/recipes/export-to-pdf)
